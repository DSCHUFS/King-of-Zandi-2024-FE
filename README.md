### 개발 동기
본 웹사이트는 Google Developer Students Clubs의 방학 프로그램 ‘잔디왕 김훕스’의 참여자들이 개발 공부 습관을 키우고 꾸준한 개발을 할 수 있도록 도와주기 위해 개발되었습니다. ‘잔디왕 김훕스’의 목적은 개발자들이 방학기간동안 매일 깃허브에 커밋을 하여 자신의 개발 역량을 향상시키는 것입니다.

이 웹사이트는 참여자들이 매일 커밋하는 시각, 연속 커밋 횟수, 총 커밋 수 등을 추척하고, 이를 통해 전체 참여자들의 랭킹을 조회할 수 있게 합니다. 이는 참여자들에게 동기 부여를 주고, 서로의 진행 상황을 확인하면서 경쟁적이면서도 협력적인 환경을 조성할 수 있도록 합니다.

이 서비스는 특히 개발에 관심이 많지만, 방학 기간동안 학습 동기를 유지하기 어려운 학생들에게 유용합니다.
